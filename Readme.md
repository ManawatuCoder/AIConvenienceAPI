# How to Run #
``` 
cd javaconnector
Build: mvn clean package
MCP Inspector command: npx @modelcontextprotocol/inspector
Path command: -jar D:/AiCon/AIConvenienceAPI/JavaConnector/target/MCP-Server-1.0.0.jar
Change the path command to where the MCP-Server.1.0.0.jar file is stored locally on your machine. Server is run locally using STDIO.

```
