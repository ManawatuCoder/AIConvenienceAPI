*Azure Blob Storage Demo*

Resources:

- Azure Rest Api Specs: https://github.com/Azure/azure-rest-api-specs
- Path: azure-rest-api-specs/specification/storage/resource-manager/Microsoft.Storage/stable/2019-06-01/blob.json
- Raw JSON path: https://raw.githubusercontent.com/Azure/azure-rest-api-specs/main/specification/storage/resource-manager/Microsoft.Storage/stable/2019-06-01/blob.json

Why This File?:

- It contains the data-plane operations (i.e., working directly with blobs, containers, etc.)

- It’s a full OpenAPI v2 spec (JSON format)

- It’s the right layer of abstraction for generating usable convenience APIs in Java

- It’s the source-of-truth used by Azure SDK engineers to generate storage client libraries

Summary Flow:

OpenAPI JSON (blob.json)
         ↓ (tsp-client convert)
TypeSpec files (.tsp)
         ↓ (tsp compile + Java emitter)
Low-level Java SDK
         ↓ (LLM + guidelines)
High-level Java convenience wrapper
